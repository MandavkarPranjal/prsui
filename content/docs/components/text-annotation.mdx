## Usage

After you add the component with the CLI, import it from your UI folder:

```tsx
"use client";

import { TextAnnotation } from "@/components/prsui/text-annotation";

export function MyComponent() {
  return (
    <TextAnnotation
      action="highlight"
      color="#ffd1dc"
      strokeWidth={1.5}
      animationDuration={600}
      iterations={2}
      padding={2}
      multiline={true}
    >
      Highlighted text
    </TextAnnotation>
  );
}
```

## Props

| Prop                | Type               | Default       | Description                                                   |
| ------------------- | ------------------ | ------------- | ------------------------------------------------------------- |
| `children`          | `React.ReactNode`  | -             | The text content to annotate.                                 |
| `action`            | `AnnotationAction` | `"highlight"` | The annotation action type.                                   |
| `color`             | `string`           | `"#ffd1dc"`   | The annotation color (CSS color string).                      |
| `strokeWidth`       | `number`           | `1.5`         | The stroke width of the annotation.                           |
| `animationDuration` | `number`           | `600`         | The duration of the animation in milliseconds.                |
| `iterations`        | `number`           | `2`           | The number of animation iterations.                           |
| `padding`           | `number`           | `2`           | The padding around the annotation.                            |
| `multiline`         | `boolean`          | `true`        | Whether the annotation supports multiline text.               |
| `isView`            | `boolean`          | `false`       | Whether to wait for the element to be in view before showing. |

## Annotation Actions

The `action` prop supports the following values:

| Action           | Description                        |
| ---------------- | ---------------------------------- |
| `highlight`      | A background highlight annotation. |
| `underline`      | An underline annotation.           |
| `box`            | A box outline annotation.          |
| `circle`         | A circle outline annotation.       |
| `strike-through` | A strikethrough annotation.        |
| `crossed-off`    | A crossed-off annotation.          |
| `bracket`        | A bracket annotation.              |
