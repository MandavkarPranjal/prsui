## Usage

After you add the component with the CLI, import it from your UI folder:

```tsx
import { Link01, Link02, Link03, Link04, Link05 } from "@/components/ui/link";

export function MyComponent() {
  return <Link01 href="https://example.com">Visit example</Link01>;
}
```

## Variants

| Variant  | Description                                                                  |
| -------- | ---------------------------------------------------------------------------- |
| `Link01` | Underline animation from left to right on hover. External `<a>` tag.         |
| `Link02` | Underline animation from right to left on hover. External `<a>` tag.         |
| `Link03` | Underline animation from center outward on hover. External `<a>` tag.        |
| `Link04` | Full-width highlight with mix-blend-difference on hover. External `<a>` tag. |
| `Link05` | Left-to-right wipe with mix-blend-difference on hover. External `<a>` tag.   |

## Props

All variants extend the native `<a>` attributes via the shared `LinkProps` type, plus two extras:

| Prop        | Type        | Default           | Description                                                                |
| ----------- | ----------- | ----------------- | -------------------------------------------------------------------------- |
| `href`      | `string`    | -                 | The link URL (passed to the underlying `<a>`)                              |
| `children`  | `ReactNode` | -                 | Link content                                                               |
| `className` | `string`    | -                 | Additional CSS classes merged onto the root `<a>`                          |
| `icon`      | `ReactNode` | arrow SVG         | Custom icon node; replaces the default arrow (add your own hover classes)  |
| `target`    | `string`    | `_blank` (Link01) | All other native `<a>` props (`target`, `rel`, `onClick`, …) are forwarded |

```tsx
import type { LinkProps } from "@/components/ui/link";

// Bring your own target / rel / onClick, etc.
<Link02
  href="https://example.com"
  target="_self"
  rel="nofollow"
  icon={<SomeIcon className="size-4" />}
>
  Visit example
</Link02>;
```
