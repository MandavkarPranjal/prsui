## Usage

After you add the component with the CLI, import it from your UI folder:

```tsx
"use client";

import { TextHighlighter } from "@/components/prsui/text-highlight";

export function MyComponent() {
  return (
    <TextHighlighter
      as="p"
      className="text-4xl font-medium"
      highlightColor="hsl(25, 90%, 80%)"
      direction="ltr"
    >
      Highlighted text
    </TextHighlighter>
  );
}
```

## Props

| Prop               | Type                                     | Default                   | Description                                              |
| ------------------ | ---------------------------------------- | ------------------------- | -------------------------------------------------------- |
| `children`         | `React.ReactNode`                        | -                         | The text content to be highlighted.                      |
| `as`               | `ElementType`                            | `"span"`                  | HTML element to render as.                               |
| `triggerType`      | `"hover" \| "ref" \| "inView" \| "auto"` | `"inView"`                | How to trigger the animation.                            |
| `transition`       | `Transition`                             | `{ type: "spring", ... }` | Animation transition configuration.                      |
| `useInViewOptions` | `UseInViewOptions`                       | `{ once: true, ... }`     | Options for useInView hook when triggerType is "inView". |
| `className`        | `string`                                 | -                         | Class name for the container element.                    |
| `highlightColor`   | `string`                                 | `"hsl(25, 90%, 80%)"`     | Highlight color (CSS color string).                      |
| `direction`        | `"ltr" \| "rtl" \| "ttb" \| "btt"`       | `"ltr"`                   | Direction of the highlight animation.                    |

## Ref Methods

The component exposes a ref with the following methods:

| Method                | Description                      |
| --------------------- | -------------------------------- |
| `animate(direction?)` | Trigger the highlight animation. |
| `reset()`             | Reset the highlight animation.   |
