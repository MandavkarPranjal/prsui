## Usage

After you add the component with the CLI, import it from your UI folder:

```tsx
"use client";

import Typewriter from "./typewriter";

export const TypewriterPreview = () => (
  <div className="flex flex-col items-center gap-2 text-center">
    <p className="text-3xl text-muted-foreground">To seek the truth we must</p>
    <p className="text-3xl font-medium text-foreground">
      <Typewriter
        as="span"
        text={[
          "question",
          "observe",
          "doubt",
          "discover",
          "keep looking upward",
        ]}
        speed={70}
        waitTime={1500}
        deleteSpeed={40}
        cursorChar={"_"}
      />
    </p>
  </div>
);
```

## Props

| Prop                      | Type                                                             | Default  | Description                                           |
| ------------------------- | ---------------------------------------------------------------- | -------- | ----------------------------------------------------- |
| `text`                    | `string \| string[]`                                             | -        | Text to type. Pass an array to cycle through phrases. |
| `as`                      | `ElementType`                                                    | `"div"`  | Element or component to render as.                    |
| `speed`                   | `number`                                                         | `50`     | Typing speed in ms per character.                     |
| `initialDelay`            | `number`                                                         | `0`      | Delay before typing starts in ms.                     |
| `waitTime`                | `number`                                                         | `2000`   | Pause before deleting or moving to next phrase in ms. |
| `deleteSpeed`             | `number`                                                         | `30`     | Deleting speed in ms per character.                   |
| `loop`                    | `boolean`                                                        | `true`   | Whether to loop through the phrases.                  |
| `className`               | `string`                                                         | -        | Classes applied to the wrapper element.               |
| `showCursor`              | `boolean`                                                        | `true`   | Whether to show the cursor.                           |
| `hideCursorOnType`        | `boolean`                                                        | `false`  | Hide the cursor while typing/deleting.                |
| `cursorChar`              | `string \| React.ReactNode`                                      | `"\|"`   | Character or node rendered as the cursor.             |
| `cursorClassName`         | `string`                                                         | `"ml-1"` | Classes applied to the cursor element.                |
| `cursorAnimationVariants` | `{ initial: Variants["initial"]; animate: Variants["animate"] }` | blinking | Animation variants for the cursor.                    |
