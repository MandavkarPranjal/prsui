## Installation

If you've already added the prsui registry, use the short command:

<CodeBlockCommand
  __bun__="bunx --bun shadcn@latest add @prsui/smooth-input"
  __npm__="npx shadcn@latest add @prsui/smooth-input"
  __pnpm__="pnpm dlx shadcn@latest add @prsui/smooth-input"
  __yarn__="yarn shadcn@latest add @prsui/smooth-input"
/>

Otherwise, use the full registry URL:

<CodeBlockCommand
  __bun__="bunx --bun shadcn@latest add https://prsui.pr5.dev/r/smooth-input.json"
  __npm__="npx shadcn@latest add https://prsui.pr5.dev/r/smooth-input.json"
  __pnpm__="pnpm dlx shadcn@latest add https://prsui.pr5.dev/r/smooth-input.json"
  __yarn__="yarn shadcn@latest add https://prsui.pr5.dev/r/smooth-input.json"
/>

## Usage

After you add the component with the CLI, import it from your UI folder:

```tsx
import { SmoothInput } from "@/components/ui/smooth-input";

export function MyComponent() {
  return <SmoothInput placeholder="Enter text..." />;
}
```

## Props

| Prop               | Type                   | Default           | Description                 |
| ------------------ | ---------------------- | ----------------- | --------------------------- |
| `type`             | `"text" \| "password"` | `"text"`          | Input type                  |
| `placeholder`      | `string`               | `"Enter text..."` | Placeholder text            |
| `value`            | `string`               | -                 | Controlled value            |
| `defaultValue`     | `string`               | -                 | Default value               |
| `wrapperClassName` | `string`               | -                 | Class for the outer wrapper |
