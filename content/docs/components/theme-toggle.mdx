## Usage

After you add the component with the CLI, import it from your UI folder:

```tsx
import { ThemeToggle } from "@/components/prsui/theme-toggle";

export function MyComponent() {
  return <ThemeToggle />;
}
```

## Next.js (next-themes)

Pass theme and onThemeChange to keep useTheme() subscribers in sync on toggle. When controlled, the component skips localStorage and lets the parent own persistence.

```tsx
"use client"

import { useTheme } from "next-themes"

import { ThemeToggle } from "@/components/prsui/theme-toggle"

export function ThemeToggleNextThemesDemo() {
  const { resolvedTheme, setTheme } = useTheme()

  return (
    <div className="flex justify-center p-6">
      <ThemeToggle
        theme={resolvedTheme === "dark" ? "dark" : "light"}
        onThemeChange={setTheme}
      />
    </div>
  )
}
```

## CSS

Add the following to your global CSS to disable the default view transition animation for the theme toggle:

```css
::view-transition-old(root),
::view-transition-new(root) {
  animation: none;
  mix-blend-mode: normal;
}
```

## Props

| Prop            | Type                                                                                    | Default    | Description                                                |
| --------------- | --------------------------------------------------------------------------------------- | ---------- | ---------------------------------------------------------- |
| `duration`      | `number`                                                                                | `400`      | Transition duration in milliseconds                        |
| `variant`       | `"circle" \| "square" \| "triangle" \| "diamond" \| "hexagon" \| "rectangle" \| "star"` | `"circle"` | Transition shape variant                                   |
| `fromCenter`    | `boolean`                                                                               | `false`    | When true, the transition expands from the viewport center |
| `theme`         | `"light" \| "dark"`                                                                     | -          | Controlled theme value (optional)                          |
| `onThemeChange` | `(theme: "light" \| "dark") => void`                                                    | -          | Called on toggle when using controlled mode                |
