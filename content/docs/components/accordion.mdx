## Usage

After you add the component with the CLI, import it from your UI folder:

```tsx
import { Accordion, type AccordionItem } from "@/components/prsui/accordion";

const items: AccordionItem[] = [
  {
    id: "item-1",
    title: "First question",
    description: "Answer to the first question.",
  },
  {
    id: "item-2",
    title: "Second question",
    description: "Answer to the second question.",
  },
];

export function MyComponent() {
  return <Accordion items={items} defaultValue="item-1" />;
}
```

## Props

| Prop            | Type                              | Default | Description                                           |
| --------------- | --------------------------------- | ------- | ----------------------------------------------------- |
| `items`         | `AccordionItem[]`                 | -       | Array of accordion items                              |
| `value`         | `string \| null`                  | -       | Controlled currently open item id                     |
| `defaultValue`  | `string \| null`                  | `null`  | Initially open item id                                |
| `onValueChange` | `(value: string \| null) => void` | -       | Callback fired when the open item changes             |
| `collapsible`   | `boolean`                         | `true`  | When false, an open item cannot be closed by clicking |
| `className`     | `string`                          | -       | Classes applied to the root wrapper                   |
| `classNames`    | `AccordionClassNames`             | -       | Override classes for individual sub-elements          |

### AccordionItem

| Field         | Type        | Description                              |
| ------------- | ----------- | ---------------------------------------- |
| `id`          | `string`    | Unique identifier for the item           |
| `title`       | `ReactNode` | Content shown in the trigger row         |
| `description` | `ReactNode` | Content revealed when the item is open   |
| `icon`        | `ReactNode` | Optional icon displayed before the title |
| `disabled`    | `boolean`   | Disables the item                        |

### AccordionClassNames

| Key           | Description                  |
| ------------- | ---------------------------- |
| `root`        | Wrapper around all items     |
| `item`        | Each accordion row container |
| `trigger`     | The clickable button row     |
| `icon`        | The icon wrapper span        |
| `title`       | The title text span          |
| `chevron`     | The chevron rotation wrapper |
| `content`     | The animated content wrapper |
| `description` | The description text div     |
