**prsui** is a [shadcn registry](https://ui.shadcn.com/docs/registry) by [Pranjal Mandavkar](https://x.com/pr5dev) — a curated collection of polished, ready-to-install components you can drop into any React project.

## Add the Registry

First, add the prsui registry to your project:

<CodeBlockCommand
  __bun__="bunx --bun shadcn@latest registry add @prsui=https://prsui.pr5.dev/r/{name}.json"
  __npm__="npx shadcn@latest registry add @prsui=https://prsui.pr5.dev/r/{name}.json"
  __pnpm__="pnpm dlx shadcn@latest registry add @prsui=https://prsui.pr5.dev/r/{name}.json"
  __yarn__="yarn shadcn@latest registry add @prsui=https://prsui.pr5.dev/r/{name}.json"
/>

## Components

The registry currently ships:

- **[Smooth Input](/docs/components/smooth-input)** - A smooth input component with a caret animation.
